# libft

## Description
Custom C library reimplementing core libc helpers for 42 projects.

## Features
- Memory/string/char helpers and linked list API.
- Bonus `t_list` operations; reusable static lib.

## Technologies Used
- C, Makefile.

## Installation
- Run `make` (or `make bonus`) to build `libft.a`.

## Usage
- Link with `-lft -L.` in your C projects.

## Example
- `cc main.c -L. -lft` after building the library.
